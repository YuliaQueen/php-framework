## PHP-framework

Мини-фреймворк на PHP (8.2).

### Инструменты и технологии

+ Dependency injection container
+ Doctrine
+ Twig

### Реализован функционал

+ Система маршрутизации
+ Система авторизации
+ Система миграций
+ Система посредников (Middleware)
+ Система подписки на события (Events)
+ Провайдеры данных
